# Cats vs Dogs Classification (SVM)

This project classifies images of cats and dogs using a Support Vector Machine (SVM).

## Dataset

Download from:
https://www.kaggle.com/datasets/tongpython/cat-and-dog

Extract into:
dataset/
├── cats/
├── dogs/

## Install

py -m pip install opencv-python scikit-learn numpy matplotlib

## Run

py svm_classifier.py

## Output

* Accuracy in terminal
* Predicted images (cat/dog)

## Result

~60–70% accuracy on small dataset.
